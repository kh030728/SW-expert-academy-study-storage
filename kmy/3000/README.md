## 2. 문제 풀이

1. 2개의 수를 입력 받을 때마다 sort를 할 경우 시간초과 발생
2. 우선순위 큐 2개를 이용하여 중간 값을 구하자
3. 1개는 오름차순, 1개는 내림차순으로 중간값을 구한다
4. 만약 내림차순 top이 오름차순의 top 보다 크다면 두 값을 교환
5. 반복하면서 중간 값을 계속 더해감
6. 이때 각 큐의 크기는 1개의 차이가 항상 나야함
