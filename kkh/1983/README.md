# 풀이 방법
-------------------------------------------
### 이 문제를 해결하기 위해서는 1차적으로 모든 학생의 성적을 알아야 할 필요가 있다.
### 성적을 모두 수행한후 원하는 성적의 학생의 점수를 기억한다.(각 총점은 중복되지 않게 입력이 들어온다.)
### 해당 학생의 index를 구하고 해당 인덱스가 상위 몇 퍼센트에 존재하는지 확인한 후 점수를 출력해준다.
#### 등수를 계산할때 인덱스 값을 그대로 사용해서는 안된다는 점을 유의하자.
